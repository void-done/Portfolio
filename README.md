Tech Stack

This portfolio is built with a focus on performance and modern design:
* **Structure:** HTML5 (Semantic)
* **Styling:** Tailwind CSS (Utility-first framework)
* **Interactivity:** JavaScript (ES6+)
* **Animations:** Custom CSS keyframes & JS scroll events
* **Hosting:** GitHub Pages & Custom Domain

Key Features
*  **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop views.
*  **Dark Mode:** Modern, developer-centric dark aesthetic.
*  **High Performance:** Fast loading speeds with minimal dependencies.
*  **Interactive UI:** "Magic Card" hover effects and scroll-velocity text.
